Gino Latorilla's dotfiles
=========================

Features
--------

- Creates backups of overwritten links in `$DOTFILE_DIR/var/backup_*`
- Unlinks and restores real files with `dotfiles --unlink`.
- Preview actions with `dotfiles --dry-run`.

Quickstart
----------

1. `git clone` this repo.
2. `cd dotfiles`.
3. `./dotfiles`
