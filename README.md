# Gino Latorilla's dotfiles with Ansible

## Requirements

- Python 3

## Quickstart

1. Clone this repo.
2. `cd dotfiles`.
3. `./bootstrap.sh`
4. `./pkg-install.sh`
5. `./install.sh`

## Normal operations

Run `./install.sh` for syncing dotfiles.

Run `./pkg-install.sh` to ensure tools are installed.
